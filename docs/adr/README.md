# Architecture Decision Records

Permanent log of significant architectural decisions.
ADRs are never deleted. Superseded ADRs are marked as such.

## Index

| ID      | Title                     | Status   |
|---------|---------------------------|----------|
| ADR-001 | Serving architecture      | Proposed |

## Template

```
# ADR-NNN: Title
## Status: Proposed | Accepted | Deprecated | Superseded by ADR-XXX
## Context: What forces this decision?
## Decision: What did we decide?
## Consequences: Positive and negative results.
## Alternatives considered: What else was evaluated?
```
