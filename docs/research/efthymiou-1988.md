# Efthymiou (1988) SCF Equations

## Reference
Efthymiou, M. "Development of SCF Formulae and Generalised Influence
Functions for Use in Fatigue Analysis." SNAME, 1988.

## Scope
Parametric equations for SCF at chord saddle, chord crown, brace saddle,
brace crown for T/Y, K, X, and KT tubular joints.

## Implementation
See src/scf_surrogate/equations/efthymiou.py

## Validity
Only valid within DNV-RP-C203 parametric ranges.
See docs/standards/dnv-rp-c203-summary.md
