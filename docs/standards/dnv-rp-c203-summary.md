# DNV-RP-C203 (2021) Key Reference

## Parametric Validity Ranges (Efthymiou equations)

| Parameter         | Symbol      | Valid Range  |
|-------------------|-------------|--------------|
| Diameter ratio    | b = d/D     | [0.2, 0.8]   |
| Chord slenderness | g = D/2T    | [10.0, 30.0] |
| Thickness ratio   | t = t/T     | [0.3, 0.8]   |
| Brace inclination | theta (deg) | [30, 90]     |
| Gap ratio (K/KT)  | z = g/D     | [-0.6, 1.0]  |

## SCF Locations
- Chord saddle (90 deg from brace plane)
- Chord crown (in brace plane)
- Brace saddle
- Brace crown

## S-N Curve D (seawater, cathodic protection)
log(N) = log(a) - m * log(dS)
m = 3.0 for dS > 52.63 MPa
m = 5.0 for dS <= 52.63 MPa
